Graduation Admission Prediction System¶

This project is created for prediction of Graduate Admissions from an Indian prescreptive. This dataset contains several parameter which are considered important during the application of master program.

These Parameter include:

  1. GRE Score (Graduate Record Examination)-{ out of 340}
  2. TOEFL Score (Test of English as Foreign Language)-{out of 120}
  3. Universtiy Rating-{out of 5}
  4. SOP (Statement of Purpose)-{out of 5}
  5. LOR (Letter of Recommendation)-{out of 5}
  6. CGPA (Cumulative Grade Point Average)-{out of 10}
  7. Research Experience-{either 0 or 1}
  8. Chance of Admit-{between 0 and 1}

Here we have applied the multiple linear regression for our model with the method of Backward Elimination.¶

Backward Elimination: In Backward Elimination we start with all the features and remove the least significant feature at each iteration which improves the performance of the model.

The summary of our model like ( Accuracy, P value, Estimated error etc.) will be presented by this method.¶

def backwardElimination(x, sl):               #function of backward elimination
    numVars = len(x[0])
    for i in range(0, numVars):
        regressor_OLS = sm.OLS(y, x).fit()
        maxVar = max(regressor_OLS.pvalues).astype(float)
        if maxVar > sl:
            for j in range(0, numVars - i):
                if (regressor_OLS.pvalues[j].astype(float) == maxVar):
                    x = np.delete(x, j, 1)
    regressor_OLS.summary()
    return x
